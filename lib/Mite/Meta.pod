=head1 NAME

Mite::Meta - Meta information about Mite objects.

=head1 SYNOPSIS

    my $meta = $obj->meta;

=head1 DESCRIPTION

This is the meta object for all L<Mite> classes.  It provides
information about Mite objects and classes.

All information is read only.

=head2 Methods

Methods will be added on an as-needed basis.

=head3 has

    my @attributes = $meta->has;

Returns a list of the object attributes associated with the class,
the things made with C<has>.

=head1 SEE ALSO

It is loosely based on L<Moose::Meta::Class> and L<Class::MOP::Class>,
but far more limited.  It is not intended to be compatible.
